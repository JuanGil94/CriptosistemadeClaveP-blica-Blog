Criptosistema de Clave Pública- Blog Educativo

Este proyecto es una blog interactivo que explica el funcionamiento de los criptosistemas, mas especifico es el sistema de clave pública, con ejemplos, ventajas, desventajas y un caso práctico implementado en JavaScript.

El blog está dividido en 5 secciones principales:

1. ¿Qué es el Criptosistema de Clave Pública
2. ¿Cómo funciona y cuáles son sus algoritmos representativos?
3. Ventajas, desventajas y diferencias con el sistema simétrico
4. Caso práctico interactivo con JavaScript, uso de RSA Simple
5. Observaciones y recomendaciones para su implementación

Se incluye una demostración funcional de un sistema de cifrado y descifrado RSA básico en JavaScript con valores educativos:

Cómo usar

1. Clona o descarga el repositorio.
2. Abre `index.html` en tu navegador.
3. Escribe un mensaje y presiona Cifrar.
4. Luego presiona Descifrar para ver el resultado.
